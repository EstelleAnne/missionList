# @beisen/mission-list

### 功能

```
任务列表展示
```

### 单组件使用方法

```
第一步：git clone git@gitlab.beisen.co:xhbisme/mission-list.git

第二步：cd mission-list

第三步：npm i

第四步：npm run dev

第五步：访问  `http://localhost:3000`
```
###项目依赖使用方法
第一步：`npm i @beisen/mission-list --save;`

第二步：`import MissionList from '@beisen/mission-list';`

参数设置见 外层 index.js
